🌐 Ceylon Fusion Restaurant Website

A modern, responsive, and interactive restaurant website built using **HTML, CSS, and JavaScript**.  
This project is designed for a premium Sri Lankan fusion dining experience with smooth animations, dynamic menu filtering, and a full booking system.

✨ Features

- 🍽️ Modern restaurant UI/UX design
- 📱 Fully responsive (mobile, tablet, desktop)
- 💱 Currency switcher (LKR / USD)
- 🍛 Interactive menu filtering system
- 👨‍🍳 Custom food plate builder (customizer)
- 🖼️ Image gallery with lightbox
- ⭐ Customer testimonial slider
- 📅 Reservation booking system
- 📩 Form validation with success modal
- 🎯 Smooth scroll animations
- 🌙 Premium dark/light section design
- 📍 Google Maps integration
- 📞 Floating contact buttons (WhatsApp, Call, Maps)

🛠️ Technologies Used

- HTML5
- CSS3 (Custom Variables + Animations)
- JavaScript (Vanilla JS)
- Font Awesome Icons
- Google Fonts
